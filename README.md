# 📊 E-Commerce Business Intelligence Analysis

## 📌 Project Overview

This Business Intelligence project analyzes E-Commerce customer and sales data to generate meaningful insights for business decision-making. Using SQL, Python, Excel, and Power BI, the project focuses on identifying sales trends, customer behaviour, and overall business performance.

---

## 🎯 Project Objectives

* Analyze overall sales performance
* Identify top customers and products
* Monitor business growth trends
* Perform customer behaviour analysis
* Create interactive Business Intelligence dashboard

---

## 🛠 Tools and Technologies Used

* SQL – Data extraction and querying
* Python – Data analysis and preprocessing
* Power BI – Business Intelligence dashboard
* Excel – Data cleaning

---

## 📁 Project Files

* dataset.csv – E-Commerce dataset
* queries.sql – SQL analysis queries
* analysis.py – Python analysis
* dashboard.pbix – Power BI BI dashboard
* README.md – Project documentation

---

## 📊 Business Intelligence Insights

* Identified top-performing products
* Found high-value customers
* Analyzed revenue trends
* Monitored business performance using dashboard

---

## 🚀 Project Outcome

Developed an interactive Business Intelligence dashboard to help understand business performance and support data-driven decision-making.
